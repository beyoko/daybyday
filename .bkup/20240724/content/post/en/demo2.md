# Markdown Demo 1

This is a demonstration of a Markdown file. Markdown is a lightweight markup language that is commonly used for formatting text on the web. It is simple, easy to read, and can be converted into various other formats like HTML.

## Headings

To create headings in Markdown, you can use hash symbols (#). The number of hash symbols represents the level of the heading. For example:

# Heading 1
## Heading 2
### Heading 3

## Emphasis

You can make text bold by surrounding it with double asterisks or underscores, and you can make it italic by surrounding it with single asterisks or underscores. For example:

**This text is bold**
__This text is also bold__

*This text is italic*
_This text is also italic_

## Lists

Markdown supports both ordered and unordered lists. To create an unordered list, you can use hyphens, plus signs, or asterisks. For example:

- Item 1
- Item 2
- Item 3

To create an ordered list, you can use numbers followed by periods. For example:

1. Item 1
2. Item 2
3. Item 3

## Links and Images

You can create links in Markdown by enclosing the link text in square brackets and the URL in parentheses. For example:

[Click here to visit Google](https://www.google.com)

You can also embed images in Markdown by using a similar syntax, but with an exclamation mark in front of the square brackets. For example:

![Markdown Logo](https://markdown-here.com/img/icon256.png)

## Code Blocks

To include code snippets or blocks in Markdown, you can use backticks. For inline code, you can use a single backtick, and for code blocks, you can use triple backticks. For example:

`print("Hello, world!")`
